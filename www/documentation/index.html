Usage

Simply reference the relevant files at the head of your page like so:

Initialitation

        	$(document).ready(function(){
               $("#calendar").dp_calendar();
            });
#calendar is the ID of the HTML Object that will be placed instead this calendar.

Settings

events_array

Signature:

Set the events to show in the calendar.

Note: In JavaScript, the month in the Date object starts from 0. So, January = 0, February = 1, March = 2â€¦. December = 11.

The Priority value is a number [1-3]

  • 1- Low
  • 2- Medium
  • 3- Urgent

The Frecuency value is a number [1-4]

  • 1- Daily
  • 2- Weekly
  • 3- Monthly
  • 4- Yearly

date_selected

Signature: 
$("#calendar").dp_calendar({ date_selected: new Date(2011,06,30) })

Set the default date preselected. Learn more about the Date class in javascript here.

Default value is the actual date.

order_by

Signature: 
$("#calendar").dp_calendar({ order_by: 1 })

Set the default order parameter.

  • 1 = Time
  • 2 = Title
  • 3 = Priority

Default 1.

show_datepicker

Signature: 
$("#calendar").dp_calendar({ show_datepicker: true })

Enable / Disable the DatePicker calendar when you click on the name of the month. Default: true.

format_ampm

Signature: 
$("#calendar").dp_calendar({ format_ampm: true })

Enable / Disable the AM/PM hour format. Default: false.

time_format

Signature: 
$("#calendar").dp_calendar({ time_format: 'H:mm' })

Set the time format. Default: 'HH:mm'.

show_sort_by

Signature: 
$("#calendar").dp_calendar({ show_sort_by: false })

Show/hide "Order by" feature. Default: true.

show_priorities

Signature: 
$("#calendar").dp_calendar({ show_priorities: false })

Show/Hide "Priorities" feature. Default: true.

show_time

Signature: 
$("#calendar").dp_calendar({ show_time: false })

Show/Hide "Time" feature. Default: true.

show_end_time

Signature: 
$("#calendar").dp_calendar({ show_end_time: false })

Show/Hide "End Time" feature. Default: true.

link_color

Signature: 
$("#calendar").dp_calendar({ link_color: 'red' })

Set the color of the normal links. Default: '#929292'.

date_range_start, date_range_end

Signature: 
$("#calendar").dp_calendar({date_range_start: new Date(2012,03, 20), date_range_end: new Date(2012,06, 20)})

Set a date range for the calendar.

Events

onChangeMonth

Signature: 
$("#calendar").dp_calendar({
onChangeMonth: function() {
		alert("onChangeMonth Triggered!");
	}
})

This event is triggered when the next and prev months buttons are clicked.

onChangeDay

Signature: 
$("#calendar").dp_calendar({
onChangeDay: function() {
		alert("onChangeDay Triggered!");
	}
})

This event is triggered when a different day is clicked

onClickMonthName

Signature: 
$("#calendar").dp_calendar({
onClickMonthName: function() {
		alert("onClickMonthName Triggered!");
	}
})

This event is triggered when the name of the month at the top of the calendar is clicked.

onClickEvents

Signature: 
$("#calendar").dp_calendar({
onClickEvents: function(data) {
		alert("onClickEvents Triggered!");
        alert("Title "+data.title);
        alert("Description "+data.description);
        alert("Time "+data.time);
        alert("Date "+data.date);
	}
})

This event is triggered when any event in the list is clicked.

Methods

setDay

Signature: 
$.fn.dp_calendar.setDay(day)

setMonth

Signature:

setYear

Signature:

Languages

You can set your own language creating a new javascript file.

This file will have all the labels translated at your language. In the js folder you will find the jquery.dp_calendar-es.js file. This is a pre made language file (Spanish). You can duplicate this, translate, and save it with other name.

To apply the language, you have to include the script after the jquery.dp_calendar.js script.
